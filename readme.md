# [matching-game.wyze.dev](https://matching-game.wyze.dev) &middot; [![Website][website-image]][website-url]

> A matching icon game with a hard mode.

## Setup

```sh
$ git clone https://github.com/wyze/matching-game.git
$ cd matching-game
$ yarn
```

## Development

```sh
$ yarn dev
```

## Build

```sh
$ yarn build
```
